# DINYA Soluções Criativas

Site institucional da DINYA, empresa de impressão 3D personalizada em Atibaia, SP.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Google Fonts (Cormorant Garamond + DM Sans)
